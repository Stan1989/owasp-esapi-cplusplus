SecureRandomBase * esapi::SecureRandomBase::createInstance ( const NarrowString &  algorithm,
const byte *  seed,
size_t  size 
) [static, protected]

Factory method to cough up an implementation. Java offers a SecureRandom(byte[]), and this overload handles it.

Analysis: since this system uses AES-256/OFB mixer for OS provided data, it is no less secure than the raw entropy bits retrieved from the operating system. That is, generating a stream using AES-256/OFB (keyed with /dev/[u]random) is *not* less secure than using /dev/[u]random or CryptGenRandom directly. Factory method to cough up an implementation. Used by getInstance and most stack based SecureRandoms
