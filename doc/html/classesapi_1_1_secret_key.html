NarrowString SecretKey::getFormat ( ) const [virtual]

Returns the name of the primary encoding format of this key, or an a reference to an empty string if this key does not support encoding. The primary encoding format is named in terms of the appropriate ASN.1 data format, if an ASN.1 specification for this key exists. When no encoding exists, a reference to the string "RAW" should be returned.
